## ▶️ Як запустити проєкт з нуля

### 1. Встановлення інструментів

- Python 3.10+
- pip (Python package manager)
- Віртуальне середовище: `venv`

### 2. Клонування репозиторію

```bash
git clone https://github.com/alezzzunia/crowdfunding_course_job.git
cd crowdfunding-analytics
```

### 3. Створення та активація віртуального середовища

```bash
python -m venv venv
source venv/bin/activate       # Linux / macOS
venv\Scripts\activate          # Windows
```

### 4. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 5. Запуск сервера

```bash
python app.py
```

### 6. Відкрити у браузері

```
http://localhost:5000
```

## 🔌 REST API ендпоінти

| Метод | URL | Опис |
|-------|-----|------|
| GET | `/api/campaigns` | Список усіх кампаній |
| GET | `/api/campaigns?category=Технології` | Фільтр по категорії |
| GET | `/api/campaigns/{id}` | Деталі однієї кампанії |
| GET | `/api/campaigns/{id}/analytics` | ML-аналіз кампанії |
| POST | `/api/campaigns` | Створити нову кампанію |
| GET | `/api/campaigns/{id}/donations` | Пожертви кампанії |
| POST | `/api/campaigns/{id}/donations` | Додати пожертву |
| GET | `/api/stats/summary` | Загальна статистика |
| GET | `/api/stats/monthly` | Збір коштів по місяцях |
| GET | `/api/stats/by_category` | Статистика по категоріях |
| POST | `/api/bot/test` | Тест бота (без Viber) |
| GET | `/api/bot/logs` | Логи бота |
| POST | `/viber/webhook` | Viber webhook |

### Приклад запиту — створення кампанії

**POST /api/campaigns**

```json
{
  "title": "Назва кампанії",
  "description": "Опис проєкту",
  "goal": 50000,
  "category": "Технології"
}
```

**Response:**

```json
{
  "id": 1,
  "title": "Назва кампанії",
  "status": "active",
  "created_at": "2026-05-18"
}
```

## 🤖 Viber-бот: налаштування

1. Зареєструвати бота на https://partners.viber.com
2. Скопіювати токен у `viber_bot.py` → `VIBER_TOKEN`
3. Налаштувати публічний URL (ngrok для локального тестування):

```bash
ngrok http 5000
```

4. Встановити webhook:

```bash
curl -X POST https://chatapi.viber.com/pa/set_webhook \
  -H "X-Viber-Auth-Token: ВАШ_ТОКЕН" \
  -d '{"url": "https://your-ngrok-url.ngrok.io/viber/webhook"}'
```

### Команди бота

| Команда | Опис |
|---------|------|
| `/топ` | Топ-5 кампаній за збором |
| `/статус` | Загальна статистика платформи |
| `/ризики` | Кампанії з підвищеним ризиком |
| `/категорії` | Розподіл по категоріях |
| `/кампанія 1` | Деталі кампанії №1 |
| `/допомога` | Список усіх команд |

## 🐛 Проблеми і рішення

| Проблема | Рішення |
|----------|---------|
| `ModuleNotFoundError` | Перевірити встановлення залежностей: `pip install -r requirements.txt` |
| Сервер не запускається | Переконатись, що всі міграції застосовано та порт 5000 вільний |
| Viber webhook не працює | Перевірити токен та публічний URL через ngrok |
| CORS помилка | Увімкнути CORS middleware у `app.py` |
| ML-модель не завантажується | Перевірити наявність навчальних даних у `database.py` |
